Detailed Description

void add_scheduled_event ( uint32_t  event)

Adds event to Scheduler.

Adds an event to the scheduler by OR'ing the events callback with the static event_scheduled variable.

Note
This is an atomic operation to avoid missing scheduled events.
Parameters
[in]Thecallback variable for the desired event to add to the scheduler.
