Returns the lowest energy mode available.

This function refers to the static lowest_energy_mode[] array to find the lowest available non-blocked energy mode.

Note
This function will never return that the EM4 energy mode is available.
Returns
Returns the lowest available energy mode (0 = EM0) as an unsigned integer.

◆ enter_sleep()

void enter_sleep ( void  )

Enters the lowest-energy available sleep mode.

This function uses the static lowest_energy_modes[] array to find the deepest sleep energy mode that is not blocked. It then enters that energy mode.

Note
This function runs atomically. This will never enter the EM4 energy state with the current implementation.

◆ sleep_block_mode()

void sleep_block_mode ( uint32_t  EM)

Blocks the sleep mode passed.

This will modify the static lowest_energy_mode[] array to block the passed sleep mode. A sleep mode should not be blocked more than 5 times- if so, the sleep mode is likely not being unblocked.

Note
This is an atomic operation. A mode should not be blocked 5 times without being unblocked. This will cause this function to fail an assert statement.
Parameters
[in]Thedesired sleep mode to block.

void sleep_unblock_mode ( uint32_t  EM)

Blocks the sleep mode passed.

This will modify the static lowest_energy_mode[] array to unblock the passed sleep mode. A sleep mode should unblocked if it has not been blocked.

Note
This is an atomic operation. If a sleep mode is unblocked and hasn't been blocked this function may fail an assert statement.
